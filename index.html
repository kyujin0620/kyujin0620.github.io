Projects

Project 01. YOLO를 활용한 교통 사고 예측 시스템

YOLOv11과 Transformer를 활용하여 실시간 영상 속 차량과 보행자를 탐지하고, 객체의 이동 경로를 기반으로 사고 위험도를 예측하는 딥러닝 기반 교통 안전 시스템을 구현했습니다.

  • 프로젝트 유형: 딥러닝 Term Project / 팀 프로젝트
  • 주제: YOLOv11과 Transformer를 활용한 교통 사고 예측 시스템
  • 핵심 목표: 사각지대 및 보행자 돌발 진입 상황을 선제적으로 예측하여 교통사고 위험 예방
  • 주요 기술: YOLOv11, Homography, BEV Mapping, Transformer, TTC, T_CPA, CPA Distance

개발 배경 및 필요성

교통사고 사망자 수는 감소하고 있지만, 보행 중 사망자 비율은 여전히 높은 비중을 차지하고 있습니다. 특히 보행자의 갑작스러운 차도 진입이나 사각지대에서 발생하는 돌발 상황은 기존 CCTV 관제 방식만으로는 선제적으로 대응하기 어렵습니다. 이에 따라 본 프로젝트에서는 영상 기반 객체 탐지와 미래 경로 예측을 결합하여 사고 위험을 사전에 판단하는 시스템을 목표로 하였습니다.

시스템 파이프라인

  • 1단계: YOLOv11을 활용하여 실시간 영상에서 차량과 보행자 탐지
  • 2단계: Homography를 이용해 원근 왜곡이 있는 영상을 BEV 좌표로 변환
  • 3단계: Transformer 모델로 과거 이동 경로를 학습하여 미래 위치 예측
  • 4단계: 객체 간 거리와 접근 속도를 기반으로 TTC, T_CPA, CPA Distance 계산
  • 5단계: 산출된 위험도를 미니맵에 시각화하여 직관적으로 표시

기존 방식과의 차별점

기존 방식은 YOLO 객체 탐지 후 Homography 기반 BEV 변환을 수행하고, 등속도 직선 운동을 가정하여 TTC를 계산하는 현재 기반 방식에 가까웠습니다. 반면 본 프로젝트는 YOLOv11과 BEV 변환 이후 Transformer를 도입하여 객체의 과거 이동 경로를 학습하고, 미래 위치를 예측한 뒤 TTC, T_CPA, CPA Distance를 기반으로 위험도를 산출하는 미래 기반 구조를 적용했습니다.

데이터 구성 및 학습

  • 공식 데이터: Kaggle 및 도로 CCTV 영상 데이터
  • 실측 데이터: 육교 위 자체 촬영 영상 및 자체 라벨링 데이터
  • 희소 데이터: YouTube 영상 기반 고위험군 시나리오
  • 수집 앵글: 하향식 및 관제 CCTV 앵글 중심
  • 타겟 클래스: 보행자, 차량, 트럭, 버스, 오토바이, 신호등, 신호 표지판
  • 데이터 규모: 총 1,096장 train/validation 데이터
  • 전처리: Roboflow를 활용한 프레임 추출 및 보행자 바운딩 박스 라벨링
  • 학습 설정: YOLOv11 기반 epoch 30회 반복 학습

YOLO 객체 검출 성능

70.8% Precision
64.8% Recall
61.3% mAP50
41.7% mAP50-95

Transformer 모델 성능

Transformer 모델은 객체의 과거 이동 궤적을 학습하여 미래 위치를 예측하도록 구성했습니다. 학습 과정에서 Training Loss와 Validation Loss를 비교하여 모델 성능을 확인했으며, 30 epoch 기준 validation loss는 약 0.2922로 나타났습니다.

TTC 계산 검증

사람 대 사람 충돌 위험 상황에서 현재 거리와 접근 속도를 기반으로 TTC를 계산했습니다. 예시 시나리오에서는 현재 거리 0.043, 접근 속도 0.251을 적용하여 TTC = 0.043 / 0.251로 계산되었고, 화면 출력 결과 약 0.17초의 위험 상황으로 나타났습니다.

최종 구현 결과

  • YOLOv11과 Homography 기반 BEV 변환을 연동하여 차량 및 보행자의 위치 좌표를 실시간으로 매핑했습니다.
  • Transformer를 활용하여 과거 20 frame의 이동 궤적을 기반으로 미래 10 frame의 위치를 예측했습니다.
  • 예측된 위치 데이터를 바탕으로 TTC 위험도를 계산하고, 위험 등급을 시각화했습니다.
  • 단순 객체 인식이 아니라 사고 예방을 위한 실시간 위험도 예측 시스템 구조를 구현했습니다.
